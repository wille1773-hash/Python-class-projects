# Final Project: Sudoku (Pygame)

What it is: A Sudoku game built with Pygame. Includes board/cell model, puzzle generator, gradient background, and reset/restart controls. I led a small team, dividing UI, board logic, and generator workstreams, then integrated and tested as a group.

How to run:
1) `pip install pygame`
2) `python sudoku.py`

- Handling mouse/keyboard events and button hover states.
- Generating valid Sudoku boards and carving puzzles.
- Coordinating a small team: scoping tasks, merging contributions, and reconciling UI/logic changes collaboratively.
